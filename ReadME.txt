This project uses express, body parser, and play-sounds.
All sound requires the use of one of the play-sounds supported players which are: 

mplayer
afplay
mpg123
mpg321
play
omxplayer
aplay
cmdmp3

Additional players can be added, see play-sounds documentations here: https://www.npmjs.com/package/play-sound


To run, go to the directory with index.js and use the following commands
npm install express
npm install body-parser
npm install play-sounds
node index.js
